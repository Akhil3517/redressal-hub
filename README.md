# Query Assistant 

An AI-powered query assistant dashboard with built-in tools for seamless portal recommendations and generating AI complaint letters.

##  Built With
- **Frontend:** React, Vite, Tailwind CSS, Radix UI, Javascript 
- **Backend:** Node.js, Express.js
- **AI Integrations:** Capabilities for AI-driven response generation.

## 📁 Project Structure

```bash
query-assistant-main
├── frontend/             # React/Vite web dashboard
│   ├── src/              # UI components, pages, hooks, and context
│   └── package.json      # Frontend dependencies
├── server/               # Node.js/Express backend 
│   ├── controllers/      # API Controllers (AI, Portal)
│   ├── routes/           # API Routing logic 
│   └── package.json      # Backend dependencies
└── README.md
```

## 💡 Core Features
- **Dynamic Portal Recommendations:** Suggests optimal resolution portals based on user queries.
- **AI Complaint Letter Generator:** Generates automated, context-aware complaint drafts to empower users.
- **Premium User Interface:** Uses Tailwind CSS and Radix UI primitives for fluid, accessible design structures.
