# Query Assistant Frontend

This is a React + Vite + TypeScript frontend for an AI-powered query assistant dashboard.

## Getting started

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm test
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Project structure

- `src` – application source code
- `public` – static assets
